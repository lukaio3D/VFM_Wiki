# T-18 vs T-16 Falchion

> Radius vs Rate - Den Rate-Fighter überlisten

## Gegner-Profil

| Eigenschaft | T-16 Falchion |
|-------------|---------------|
| **Typ** | Rate Fighter |
| **Stärke** | Beste Sustained Turn Rate, guter Energieerhalt |
| **Schwäche** | Mittlerer Radius, kein extremes High-AoA |
| **Corner Speed** | 380-420 kts |
| **Gefährlichkeit** | Hoch bei Speed in Two-Circle! |

::: tip DEINE CHANCE
Die T-16 ist dein **machbarster Gegner** - wenn du One-Circle durchsetzt!
:::

---

## Die Strategie

### Das Kern-Prinzip

> **One-Circle erzwingen. Langsam kämpfen. Nose Authority nutzen!**

```mermaid
flowchart TD
    FIGHT[T-16 Begegnung] --> GOAL[Dein Ziel:]

    GOAL --> ONE[One-Circle erzwingen]
    GOAL --> SLOW[Kampf unter 300 kts]
    GOAL --> NOSE[Deine Nose Authority nutzen]

    ONE --> RADIUS[Dein Radius < ihr Radius]
    SLOW --> WEAK[Dort ist ihre Rate egal]
    NOSE --> SHOT[Du bekommst Nase zuerst auf Ziel]
```

## Der Kampf der Philosophien

### Rate vs Radius erklärt

```mermaid
flowchart TD
    FIGHT[Kampf] --> TYPE{Kampf-Typ}

    TYPE --> TWO[Two-Circle]
    TYPE --> ONE[One-Circle]

    TWO --> RATE[Rate entscheidet]
    RATE --> T16WIN[T-16 gewinnt!]

    ONE --> RAD[Radius entscheidet]
    RAD --> T18WIN[DU gewinnst!]
```

::: danger KRITISCH
Two-Circle = T-16 gewinnt!
One-Circle = DU gewinnst!
:::

## Phase 1: Der Merge

### Was die T-16 will

- Two-Circle-Fight (Nase auf Heck)
- Hohe Speed (380-420 kts)
- Sustained Turn nutzen
- Dich "out-raten"

### Deine Antwort

```mermaid
flowchart TD
    MERGE[Am Merge] --> ACTION[SOFORT One-Circle!]

    ACTION --> HARD[Maximaler AoA Turn]
    HARD --> NOSE[Nase auf Gegner]
    NOSE --> THREAT[Snapshot-Drohung]

    THREAT --> CHOICE{T-16 muss wählen:}
    CHOICE --> ACCEPT[Akzeptiert One-Circle]
    CHOICE --> ESCAPE[Versucht Two-Circle]

    ACCEPT --> WIN[Du gewinnst!]
    ESCAPE --> CHASE[Du jagst sie]
```

**Am Merge:**
1. **Maximaler Turn** - Nase sofort auf Gegner
2. Zeige, dass du One-Circle willst
3. Bereit für Snapshot
4. **Nicht aufhören zu drehen!**

flowchart TD

## Phase 3: Der Slow-Speed-Fight

### Warum langsam = gut für dich

| Speed | T-18 Performance | T-16 Performance |
|-------|-----------------|------------------|
| 400 kts | OK | **Beste Rate** |
| 300 kts | Gut | Gut |
| 250 kts | **Optimal!** | Schwächer |
| 200 kts | Noch steuerbar | Probleme |

### Die Mathematik

```mermaid
flowchart TD
    SLOW[Unter 300 kts] --> COMPARE[Vergleich:]

    COMPARE --> T18[T-18: Nose Authority ++]
    COMPARE --> T16[T-16: Rate sinkt]

    T18 --> DOMINATE[Du dominierst!]
    T16 --> STRUGGLE[Sie kämpft]
```

## Phase 4: Schusslösung

### Dein Vorteil: Nose Authority

Im One-Circle unter 300 kts:

```mermaid
flowchart TD
    FIGHT[One-Circle < 300 kts] --> NOSE[Deine Nase ist schneller]
    NOSE --> FIRST[Du zielst zuerst]
    FIRST --> SHOT[Snapshot!]
    SHOT --> WIN[Treffer!]
```

**Schritt für Schritt:**
1. Bleibe unter 300 kts
2. Maximaler AoA
3. Nase auf Gegner führen
4. **Snapshot** sobald möglich
5. Nicht warten auf perfekte Lösung!

## Notfall: Du bist zu schnell

Wenn du über 350 kts bist:

::: warning GEFAHR
Über 350 kts spielt die T-16 DEIN Spiel nicht mehr!
:::

### Lösung: Speed abladen

```mermaid
flowchart TD
    FAST[Zu schnell > 350 kts] --> BLEED[Speed abladen!]

    BLEED --> HOW{Wie?}
    HOW --> HARD[Harter Turn]
    HOW --> BRAKE[Airbrake]
    HOW --> VERTICAL[Kurz hochziehen]

    HARD --> SLOW[Speed sinkt]
    BRAKE --> SLOW
    VERTICAL --> SLOW

    SLOW --> OPTIMAL[Zurück zu 250 kts]
    OPTIMAL --> DOMINATE[Dominanz!]
```

## Defensive Situation

Wenn die T-16 hinter dir ist:

### Scissors einleiten

```mermaid
flowchart TD
    DEFENSE[T-16 hinter dir] --> SCISSORS[Scissors!]

    SCISSORS --> REVERSE[Harter Richtungswechsel]
    REVERSE --> SLOW[Beide werden langsam]
    SLOW --> ADVANTAGE[Dein Vorteil!]
    ADVANTAGE --> NOSE[Deine Nose Authority]
    NOSE --> REVERSAL[Rollentausch!]
```

**Warum Scissors funktioniert:**
1. Beide Flugzeuge verlangsamen
2. Bei niedriger Speed hast du Vorteil
3. Dein Radius ist kleiner
4. Du kannst die Rolle tauschen

## Zusammenfassung

### DO's (Machen!)

- One-Circle erzwingen
- Langsam kämpfen (250-300 kts)
- Maximaler AoA nutzen
- Nase auf Gegner halten
- Snapshots nehmen

### DON'TS (Vermeiden!)

- Two-Circle (= Niederlage)
- Schnell werden (> 350 kts)
- Ihr Spiel spielen
- Separation zulassen

```mermaid
flowchart TD
    VS16[vs T-16] --> RULE1[One-Circle!]
    RULE1 --> RULE2[Langsam kämpfen!]
    RULE2 --> RULE3[Nase auf Gegner!]
    RULE3 --> RULE4[Snapshot!]
    RULE4 --> WIN[Sieg!]
```

::: info MERKE
Die T-16 ist der Rate-König. Aber Rate ist egal, wenn der Kampf langsam ist. Erzwinge DEIN Spiel!
:::
