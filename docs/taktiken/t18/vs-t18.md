# T-18 vs T-18 Spiegelkampf

> Gleiche Monster - Wer blinzelt zuerst?

## Die Herausforderung

| Parameter | Deine T-18 | Gegner T-18 |
|-----------|-----------|-------------|
| Turn Radius | Identisch | Identisch |
| Nose Authority | Identisch | Identisch |
| High-AoA | Identisch | Identisch |
| **Unterschied** | **Pilot-Skill** | **Pilot-Skill** |

::: warning BRUTALER KAMPF
Zwei T-18 im Nahkampf sind wie zwei Messerstecher in einer Telefonzelle. Schnell und blutig!
:::

---

## Die Strategie

### Das Kern-Prinzip

> **Erster Schuss gewinnt. Nose first. Kein Zögern!**

```mermaid
flowchart TD
    MIRROR[T-18 Spiegelkampf] --> REALITY[Realität:]

    REALITY --> SAME[Identische Fähigkeiten]
    REALITY --> FAST[Schnelle Entscheidung]
    REALITY --> FIRST[Wer zuerst trifft]

    SAME --> SKILL[Pilot entscheidet]
    FAST --> SNAP[Snapshot-Kompetenz]
    FIRST --> WIN[Gewinnt!]
```

## Die Natur des Kampfes

### Warum es schnell geht

```mermaid
flowchart TD
    START[Merge] --> BOTH[Beide drehen sofort]
    BOTH --> NOSES[Beide Nasen auf Gegner]
    NOSES --> GUNS[Beide in Schussposition]
    GUNS --> RESULT[In Sekunden entschieden]
```

**Typischer Ablauf:**
1. Merge bei ~250-300 kts
2. Beide ziehen maximalen AoA
3. Beide bekommen Nase auf Gegner
4. Wer zuerst trifft, gewinnt
5. Oft in **einem Turn** entschieden!

## Entscheidende Faktoren

### 1. Merge-Qualität

Der Merge ist ALLES im T-18 Spiegelkampf:

```mermaid
flowchart TD
    MERGE[Am Merge] --> FACTORS{Entscheidend:}

    FACTORS --> SPEED[Speed]
    FACTORS --> ANGLE[Winkel]
    FACTORS --> TIMING[Timing]

    SPEED --> OPTIMAL["250-300 kts = Optimal"]
    ANGLE --> FACE["Nose-to-Nose bevorzugt"]
    TIMING --> LEAD["Lead Turn = Vorteil"]
```

### 2. Lead Turn

Wer zuerst dreht, hat Vorteil:

```mermaid
flowchart LR
    LEAD[Lead Turn] --> EARLY[Früher drehen]
    EARLY --> ANGLE[Winkel-Vorteil]
    ANGLE --> NOSE[Nase zuerst auf Ziel]
    NOSE --> SHOT[Erster Schuss]
```

**Aber Vorsicht:**
- Zu früh = du zeigst Heck
- Timing muss stimmen!

### 3. Guns Tracking

```mermaid
flowchart TD
    TRACK[Guns Tracking] --> SMOOTH[Smooth Inputs]
    SMOOTH --> LEAD2[Lead richtig berechnen]
    LEAD2 --> BURST[Kurze Bursts]
    BURST --> HIT[Treffer!]
```

## Taktische Optionen

### Option 1: Aggressiver Ansatz

```mermaid
flowchart TD
    AGGRO[Aggressiv] --> MERGE[Am Merge]
    MERGE --> HARD[Maximaler Turn]
    HARD --> NOSE[Nase auf Gegner]
    NOSE --> FIRE[SOFORT feuern]
    FIRE --> RESULT{Ergebnis}

    RESULT --> HIT[Treffer = Sieg]
    RESULT --> MISS[Miss = Scissors]
```

**Wann nutzen:**
- Du bist confident
- Guter Merge-Winkel
- Gegner zögert

### Option 2: Reaktiver Ansatz

```mermaid
flowchart TD
    REACT[Reaktiv] --> WATCH[Gegner beobachten]
    WATCH --> WAIT[Seinen Move sehen]
    WAIT --> COUNTER[Counter-Move]
    COUNTER --> EXPLOIT[Seinen Fehler nutzen]
```

**Wann nutzen:**
- Gegner ist aggressiv
- Schlechter Merge-Winkel
- Unsichere Situation

### Option 3: Scissors

Wenn keiner trifft:

```mermaid
flowchart TD
    MISS[Beide verfehlen] --> SCISSORS[Scissors beginnt]
    SCISSORS --> REVERSAL[Richtungswechsel]
    REVERSAL --> PATIENCE[Geduld...]
    PATIENCE --> OVERSHOOT[Wer overshoots?]
    OVERSHOOT --> SHOT[Schuss auf Overshooting Gegner]
```

## Häufige Fehler

### Fehler 1: Zu früh schießen

::: danger VERSCHWENDETE MUNITION
Wenn du zu früh schießt:
- Keine Treffer
- Munition verschwendet
- Gegner weiß wo du bist
:::

**Lösung:** Warte auf Schusslösung!

### Fehler 2: Zu spät schießen

::: danger VERPASSTE CHANCE
Wenn du zu lange wartest:
- Er schießt zuerst
- Du stirbst
:::

**Lösung:** Snapshot nehmen wenn möglich!

### Fehler 3: Vorhersagbare Moves

::: warning NICHT VORHERSAGBAR SEIN
Im Spiegelkampf:
- Variiere deine Richtungswechsel
- Überrasche mit Timing
- Sei unberechenbar
:::

## Speed-Management

### Die Speed-Zone

| Speed | Status | Bedeutung |
|-------|--------|-----------|
| 250-300 kts | Optimal | Beste Nose Authority |
| 200-250 kts | Gut | Noch voll steuerbar |
| 150-200 kts | Gefährlich | Beide fast stehend |
| < 150 kts | Kritisch | Wer zuerst Speed hat |

### Das Langsamkeits-Dilemma

```mermaid
flowchart TD
    BOTH[Beide sehr langsam] --> PROBLEM[Problem:]
    PROBLEM --> NEITHER[Keiner kann viel tun]
    PROBLEM --> WAITING[Warten auf Speed]

    NEITHER --> STALL[Quasi-Stillstand]
    WAITING --> FIRST[Wer zuerst beschleunigt]

    STALL --> PATIENT[Geduld gewinnt]
    FIRST --> RISK[Zeigt Heck = Risiko]
```

## Die Entscheidungsmatrix

### Aggressiv sein wenn:

| Situation | Aktion |
|-----------|--------|
| Du hast Lead Turn | Pressen! |
| Besserer Merge-Winkel | Sofort Nase drauf |
| Gegner zögert | Ausnutzen! |

### Defensiv sein wenn:

| Situation | Aktion |
|-----------|--------|
| Schlechter Merge | Scissors vorbereiten |
| Gegner hat Lead | Counter vorbereiten |
| Unklare Situation | Beobachten |

## Zusammenfassung

### DO's (Machen!)

- Lead Turn anstreben
- Snapshot wenn möglich
- Smooth Tracking
- Schnelle Entscheidungen

### DON'TS (Vermeiden!)

- Zu früh schießen
- Zu lange warten
- Vorhersagbar sein
- Zögern am Merge

```mermaid
flowchart TD
    MIRROR[T-18 Spiegelkampf] --> RULE1[Lead Turn!]
    RULE1 --> RULE2[Nase zuerst!]
    RULE2 --> RULE3[Snapshot!]
    RULE3 --> RULE4[Keine Gnade!]
    RULE4 --> WIN[Sieg!]
```

::: info MERKE
Im T-18 Spiegelkampf gibt es keine zweite Chance. Wer zuerst die Nase auf Ziel hat und trifft, gewinnt. Sei schnell, sei präzise, sei gnadenlos!
:::
